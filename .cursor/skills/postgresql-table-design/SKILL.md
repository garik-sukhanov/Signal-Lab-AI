# PostgreSQL Table Design

## When to Use
- Design new tables or relations.
- Review indexing and constraints.
- Validate schema changes for performance and maintainability.

## Focus
- Correct data types and constraints.
- Index strategy for read/write balance.
- Backward-compatible migrations and query performance.
